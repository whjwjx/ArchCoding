## 五、排版与 PDF

### HTML 模板

采用 layout.md 定义的排版规范：
- **配色**：紫色主色调（#6B4C9A）+ 橙色强调色（#E67E22）
- **字体**：PingFang SC / Microsoft YaHei
- **字号**：正文 15px、章节标题 22px
- **间距**：页边距 60px、段落间距 16px

### PDF 生成

使用 Chrome Headless 生成 PDF：
```
chrome.exe --headless=new --print-to-pdf="产品架构师养成手册.pdf"
```
