## 📝 核心方法速查

### SMART提问法
- **S**pecific（具体的）
- **M**easurable（可衡量的）
- **A**chievable（可实现的）
- **R**elevant（相关的）
- **T**ime-bound（有时限的）

### 5W2H分析法
- **What**（什么）
- **Why**（为什么）
- **Who**（谁）
- **When**何时）
- **Where**（何地）
- **How**（如何）
- **How much**（多少）

### 提示词公式
```
角色设定 + 任务描述 + 上下文信息 + 输出格式 + 具体要求
```
